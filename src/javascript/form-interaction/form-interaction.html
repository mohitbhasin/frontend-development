Interacting with forms

Common form elements

<input>: support various fields with types include text, password, checkbox, email, date, radio, submit

<textarea>: allows text for longer and multiline input.

<select>: dropdown with a list of options.

<button>: clickable button that can trigger custom js functions, or used to submit/reset a form.

<label>: text caption that can be associated with specific form control
