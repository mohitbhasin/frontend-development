Working with classes

  • Every element p, h, div are javascript objects.
  • They belong to HTMLElement family.
  • All elements inherit a property called classList.
  • classList property returns DOMTokenList, which contains .toggle(), .add(), .remove() etc

ClassList for this text includes:

clasList.toggle automatically assigns or reamoves a class

A class to toggle can be picked up from css or <style>. Common use cases include setting dark-mode class on the <body>, expandind or collapsing menus.

Commonly used DOM properties: innerHTML, textContent, style

  • textContent: gets or sets text content without affecting HTML.
  • style: accesses inline style of an element.
