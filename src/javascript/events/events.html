Event Delegation

Event listener can be used to track events of children elements. It relies on Event Bubbling. When an event happens it travels up the DOM tree to parent, grandparent and so on until document object.

  • Square
  • Circle
  • Triangle
  • Rectangle

Event object's important properties include:

  • target: Element where the event started
  • currentTarget: Owner element where the listener is attached
  • type: type of event - 'click', 'submit', 'keydown' or 'mouseover'
  • bubbles: Boolean indicating if an event bubbles up to DOM
  • timeStamp: time elapesed (in miliseconds) after page loading and the event to happen
