JSON objects

Json data types include:

  • String: e.g. "Javascript"
  • Numbers: including integers, decimals and scientific notations(exponent) e.g. 543, 3.141, 5e+10
  • Boolean: e.g. true or false
  • Arrays: e.g. [32, "Javascript", 3.14]
  • objects: e.g. {fruit: [orange, banana, mango]}
  • null
