Dynamically managing DOM elements

Append vs Append Child

append() can be used to add an element in Node object form. Or add a DOMString (text).

.append() accepts multiple argument.

// append(Node_object) for parent <ul>.append(<li>) converts to:
// <ul><li>...</li></ul>

// append(String) for parent <ul>.append("Banana") converts to:
// <ul>Banana></ul>
    Items:

appendChild() accepts single argument of type Node object.

Similarly, there are remove() and removeChild() methods to remove objects and DOMString.
