Introduction to TypeScript

  • Superset of JS
  • Every js code/file is already a valid TypeScript code/file.
  • It is statically typed language that compiles to plain javascript.

TypeScript setup

  • Install TypeScript compiler
    • // Installing locally for the project
// This will create node_modules directory, package.json and package-lock.json
npm install --save-dev typescript
  • Initialize config
    • // Since typeScript is not installed globally, use npx to call tsc
// This will create tsconfig.json file in which you specify how to organize compiled files (.map, .d.ts, etc).

npx tsc --init
  • Update tsconfig.json to sepatate the source(what you write) from distribution(what browser runs).
    • "rootDir": "./src"
"outDir": "./dist"
"target": "ES6", // or higher(ES2020)
"module": "ES6", // crucial

// Make sure to create src/ directory and place .ts file in it.
// The dis/ folder will be created automatically.
  • Create and load the typescript module
    • // The script must be loaded as a module.
// Point to the .js version of your typescript (usually in ./dist folder).
    
<script type="module" src="../typescript/dist/introduction.js"></script>
  • Start the compiler watch mode
    • npx tsc -w
